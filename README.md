# cinema
Cinema/Movie Theater system exploring Microservices using Nest.js framework and Kafka.

## Creating a new service

### Create service project

```
nest new <name-of-the-service>
```

### Add basic dependencies

```
yarn add @nestjs/config @nestjs/microservices kafkajs @nestjs/swagger swagger-ui-express class-validator class-transformer
```

### Add MongoDB dependencies

```
yarn add @nestjs/mongoose mongoose
```
